# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability, please use GitHub Private Vulnerability Reporting via the repository’s Security tab.

We will review the report and respond as soon as reasonably possible.

Please include:

* a description of the vulnerability
* steps to reproduce
* affected versions or commits, if known
* possible impact
* any suggested mitigation, if available

Please avoid publicly disclosing the vulnerability until we have had a chance to investigate and, if necessary, release a fix.
